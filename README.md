基于[ddz_game](https://github.com/tinyshu/ddz_game)项目的服务端改的，ddz_game项目服务端是nodejs的，把控不了
，取消了数据库的依赖，纯内存

## 运行
1. 安装java 
2. 运行GameServer的main函数
3. 客户端用cocos creator运行ddz_game的client项目
